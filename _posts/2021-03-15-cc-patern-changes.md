이렇게 Header와 Header 컴포넌트 안에 있는 자식의 컴포넌트들을 children props를 이용해 정의해줌으로서 아래와 같이 간단하면서도 가독성있는 컴포넌트를 만들 수 있다.

```react
function Head() {
  return (
    <Header>
      <Header.Logo />
      <Header.MidWrapper>
        <Header.Input />
        <Header.SearchButton />
      </Header.MidWrapper>
      <Header.BottomWrapper>
        <Header.SignInButton>Sign in</Header.SignInButton>
      </Header.BottomWrapper>
    </Header>
  )
}
```
